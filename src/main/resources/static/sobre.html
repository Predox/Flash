Sobre o Editor de Imagens

Este site foi desenvolvido com o objetivo de oferecer uma alternativa simples, rápida e eficiente para edição de imagens diretamente no navegador. Com uma interface intuitiva e recursos práticos, você pode aplicar efeitos, ajustar configurações e salvar suas imagens com facilidade.

O projeto foi desenvolvido por Pedro Henrique Haupt, com foco em acessibilidade, desempenho e integração com banco de dados. Ideal para quem busca uma solução leve e funcional para edições básicas.

Tecnologias utilizadas:

  • Java (Spring Boot) no backend
  • HTML, CSS e JavaScript no frontend
  • MySQL para armazenamento de dados
